# coding.md

This includes plugins that changes how you code, how you interact with code (or
just specific file types) and such.

## bullets

Automatic bullet points like in Obsidian or Google Docs

## inc-rename

When renaming something using LSP, you instead will see it live change with your
eyes, waow!

## lazydev

Just adds dev stuff to neovim and lazy.nvim related lua files

## neogen

Allows for auto creation of annotations like lua types and stuff

## new-file-template

Allowes you to define templates for new files using lua

- Check out `lua/templates` for examples and the ones I've already written

## ts-autotag

Adds features relating to automatic HTML tags

## ts-comments

Adds tree sitter aware comments or something (like inline codeblocks) and allows
for setting easy custom comment strings
